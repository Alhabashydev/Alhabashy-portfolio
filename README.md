# Moustafa Alhabashy Portfolio

A production-ready, single-page, black/white 3D animated developer portfolio built with React, Vite, TypeScript, Tailwind CSS, Framer Motion, React Three Fiber, Drei, Lucide React, and React Icons.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. Open Vercel and import the repository.
3. Framework preset: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.
6. Deploy.

## Edit content

Most content is in `src/data/`:

- `skills.ts`
- `technologies.ts`
- `services.ts`
- `projects.ts`
- `timeline.ts`
- `whyWorkWithMe.ts`

Update links in `Contact.tsx`, `ContactCTA.tsx`, and `Footer.tsx`.
